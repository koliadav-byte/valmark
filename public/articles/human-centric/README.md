Категория: Human Centric

Положите сюда папки публикаций (по одной папке на статью). Каждая папка должна содержать:
- `article.pdf` — основной PDF
- `summary.md` — краткая управленческая выжимка
- опционально `metadata.json` с полями: title, author, date (ISO)

Пример:
human-centric/experience-design/
  - article.pdf
  - summary.md
  - metadata.json
